# 🎨 AirCanvas-AI

An AI-powered virtual drawing application that lets you draw in the air using hand gestures via your webcam.

Built using MediaPipe and OpenCV, this project demonstrates real-time hand tracking, gesture recognition, and computer vision interaction.

---

## 🚀 Features

- ✋ Hand tracking using MediaPipe
- 🎨 Draw in air using index finger
- ✌️ Gesture control (2 fingers = selection mode)
- 🤏 Pinch to adjust brush thickness
- 🎯 Multiple colors + eraser
- 🧹 Clear canvas option
- 💾 Save drawings locally

---

## 🧠 Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy
